# Repository Conduct

Working together on this codebase requires standard professional courtesy. I welcome contributors from various places, and treating everyone with respect is a hard requirement to participate here.

Keep your PR reviews and Issue comments focused heavily on architecture, logic, and code quality. Ad hominem remarks, exclusionary language, or personal attacks will result in an immediate block from the repository.

If a technical debate escalates, pause the thread. Give the maintainers time to review the arguments and make the final technical call.

Should you encounter behavior that violates these boundaries, please email me. I don't do public moderation debates. Disruptive users simply get blocked.

*(Based on the [Contributor Covenant](https://www.contributor-covenant.org) v1.4)*
